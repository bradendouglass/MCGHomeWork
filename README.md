## Fairly classic app

1. Clone repo
1. Run bundle install or with binstubs if that is how you role
1. Change db/seeds.rb to auto create your dealer name
1. rake db:migrate && db:seed
1. Rails s
1. Your favorite web browser at localhost:3000
